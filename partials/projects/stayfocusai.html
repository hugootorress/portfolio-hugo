Descripción del Proyecto

StayFocusAI es una herramienta de productividad avanzada que utiliza Visión Artificial para ayudar a desarrolladores a mantener el enfoque. A diferencia de las apps tradicionales, StayFocusAI corre en segundo plano y utiliza una red neuronal para detectar distracciones físicas (uso del móvil), emitiendo alertas audibles y registrando datos para su posterior análisis estadístico.

Características Principales

Inferencia de IA

Uso de YOLOv8 (You Only Look Once) para la detección de objetos con alta precisión y baja latencia.

Modo Invisible

Integración con la bandeja del sistema (System Tray) mediante PyStray para operar sin ventanas abiertas.

MLOps & Data Log

Sistema de guardado de evidencias visuales y logs en CSV para el re-entrenamiento y auditoría de enfoque.

Alertas Asíncronas

Notificaciones de voz multihilo (gTTS/pyttsx3) y notificaciones nativas de Windows para corrección inmediata.

Configuración YAML

Arquitectura desacoplada que permite ajustar umbrales y tiempos mediante archivos de configuración externos.

Contenerizado

Dockerizado para garantizar la portabilidad de las dependencias de Computer Vision.

Tecnologías Utilizadas

Core & AI

  • Python 3.11: Lenguaje base del sistema
  • YOLOv8 (Ultralytics): Modelo de Deep Learning
  • OpenCV: Procesamiento de frames de video
  • PyTorch: Framework de computación tensorial

Data & System

  • Pandas & Seaborn: Análisis y visualización de datos
  • PyYAML: Gestión de configuración jerárquica
  • PyStray & Plyer: Integración con el SO y notificaciones
