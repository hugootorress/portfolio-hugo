Descripción del Proyecto

TallerMate es una aplicación web Full Stack diseñada para la gestión integral de talleres mecánicos. La aplicación permite administrar de manera eficiente clientes, vehículos, reparaciones, inventario y facturación, todo desde una interfaz moderna e intuitiva.

Características Principales

Autenticación Segura

Sistema de autenticación con Laravel Sanctum y gestión de tokens JWT

Gestión de Roles

Roles de usuario diferenciados (Administrador y Mecánico) con permisos específicos

Generación de Facturas

Creación automática de facturas en formato PDF con todos los detalles de la reparación

Dashboard Estadístico

Panel de control con métricas en tiempo real sobre el estado del taller

Dockerizado

Aplicación completamente dockerizada para fácil despliegue en cualquier entorno

Responsive Design

Diseño adaptable que funciona perfectamente en escritorio, tablet y móvil

Tecnologías Utilizadas

Backend

  • Laravel 10: Framework PHP para la API RESTful
  • Laravel Sanctum: Autenticación de API con tokens
  • MySQL: Base de datos relacional
  • Docker: Contenedorización de la aplicación

Frontend

  • Angular 16: Framework para SPA
  • TypeScript: Lenguaje de programación tipado
  • RxJS: Programación reactiva
  • Angular Material: Componentes UI

Despliegue

AWS

La aplicación fue desplegada en Amazon Web Services utilizando EC2 para hosting, S3 para almacenamiento de archivos, y RDS para la base de datos relacional.
