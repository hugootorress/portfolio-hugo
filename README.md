# Portfolio Hugo Torres

Portafolio personal desarrollado con HTML, CSS y JavaScript vanilla. Incluye formulario de contacto integrado con Netlify.

## Tecnologías

- **HTML5**: Estructura del sitio con partials cargados dinámicamente
- **CSS3**: Estilos modularizados en varios archivos (base, componentes, responsive)
- **JavaScript (ES6 Modules)**: Funcionalidad front-end organizada en módulos independientes
- **Netlify**: Hosting y procesamiento de formularios
- **Font Awesome**: Iconografía

## Características

- Formulario de contacto con validación y notificaciones
- Menú móvil responsive
- Scroll suave entre secciones
- Carga dinámica de partials HTML
- Modularización de código JavaScript
- Integración con Netlify Forms
